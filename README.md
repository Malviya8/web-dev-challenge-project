Welcome to the official repository for the **Robotics Club of NIT Patna**. This project is a high-performance, responsive landing page designed to showcase our club's innovations, events, and technical team.

It is built entirely with **Vanilla JavaScript, HTML5, and CSS3**, featuring a custom-built physics engine for the background without relying on heavy external frameworks.

## ‚ú® Key Features

### üé® Visuals & UI
* **Interactive Particle Engine:** A custom HTML5 Canvas background where nodes connect dynamically based on proximity and react to mouse movements.
* **Glassmorphism Design:** Modern, translucent card aesthetics with backdrop-blur effects used for the Navbar, Project cards, and Team sections.
* **Neon Breathe Preloader:** A custom CSS animation that creates a glowing "breathing" effect on the logo before the site loads.

### ‚ö° Performance & Interactivity
* **Zero Framework Dependency:** Pure Vanilla JS ensures lightning-fast load times.
* **Dynamic Typewriter Effect:** An infinite loop typing animation ("Design. Build. Innovate.") in the hero section.
* **Scroll-Reveal Animations:** Elements fade in and slide up as you scroll down using the `IntersectionObserver` API.
* **Responsive Layout:** Fully optimized for mobile (hamburger menus hidden), tablets, and desktops.
